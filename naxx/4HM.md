|cyellowFour Horsemen

{triangle} Zeliek: C'thun beam. NO MELEE DPS AT ALL. /range 10
{circle} Mograine: Frontal Fire dmg.
{star} Korth'Azz: Meteor. Stack on tank. Move in/out as a group.

{tank} Tank, then Safe Spot: Boss 1 -> Safe Spot -> Boss 2 -> Safe Spot -> Repeat
{tank} You will only tank 2 bosses.
{tank} Front Group (~150+FR) ({circle}Mograine, {star}Korth'azz): Yits ({star}), Pi ({circle}), Log (safe - {star} first), Wilkey (safe - {circle} first)
{tank} Back Group ({diamond}Blameux, {triangle}Zeliek): Natuur ({diamond}), Braiking ({triangle}), Dritzz (safe - {diamond} first), Axetwomouth (safe - {triangle} first)

{dps} Move Clockwise: Mograine -> Korth -> Safe Spot -> Repeat
{dps} DPS Grp A [Start {circle}Mograine]
{dps} DPS Grp B [Start {star}Korth'azz]
{dps} DPS Grp C [Start Safe Spot]
{dps} Once a horseman dies, rotate to others. Keep on 3-mark cycle. You will spend a lot of time in the safe zone. Bandage.

{healer} Move Clockwise, in a square (No safe spot!) Blameux -> Zeliek -> Mograine -> Korth'Azz -> Repeat
{healer} Healer Grp 1 (Move Mark 1) Crayonz, Dicadactle, Boogerz, Sjue
{healer} Healer Grp 2 (Move Mark 2) Tasteh, Jaiyla, Zindiyas, Gabmucs
{healer} Healer Grp 3 (Move Mark 3) Drgoo, Audible, Vijay, Kordesh

{healer} Start {triangle}Zeliek: Dicadactle, Jaiyla, Audible
{healer} Start {circle}Mograine: Boogerz, Zindiyas, Ccow
